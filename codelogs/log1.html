Let's say the assignment needs you to include some sample code. Here's how to do it: use the <textarea> tag.

Other Tags You Can Use

As you are writing your response, feel free to use lists, tables, and other HTML elements:

  • Lists
  • Tables
  • Headlines like h1, h2, etc.
  • Paragraphs
  • Any other HTML tags
  • Horizontal rules like hr

You can also add as much CSS as you want.

Finally, if you want to simulate the appearance of a tag without using the tag, use the &lt; and &gt; characters to show < and >.
